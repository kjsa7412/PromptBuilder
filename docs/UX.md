# UX 명세 (UX)

## 사용자 여정 (User Journey)

### 비로그인 사용자
1. 메인(/) → 주간 인기/랜덤/신규 카드 탐색
2. 카드 클릭 → 상세(/p/[id])
3. Prompt Builder에서 변수 입력
4. Generate → 완성 프롬프트 확인
5. Copy 또는 "Chat UI로 사용"

### 로그인 사용자
1. /login → Supabase Auth 로그인
2. 상세 페이지 → 북마크 클릭
3. /me/library → 저장된 프롬프트 목록
4. /me/prompts/new → 새 프롬프트 작성
5. 변수 정의 → 미리보기 → 발행

## 핵심 UX 원칙

- **즉시성**: Generate 클릭 → 0.5초 이내 결과 (프론트 치환)
- **명확성**: 필수 변수 미입력 시 명확한 에러 표시
- **접근성**: 로그인 없이 핵심 기능 체험 가능
- **공유**: URL 공유로 동일 프롬프트 접근

## 페이지별 UX

### 메인 (/)
- Hero: 서비스 한 줄 소개 + 탐색 CTA
- 주간 인기 (trending): 최대 6개 카드
- 랜덤 발견: 6개 카드, "더 보기" 버튼
- 신규: 최신 6개
- 태그 클라우드: 인기 태그 클릭 시 /explore?tag= 이동

### 탐색 (/explore)
- 검색바 (제목/태그 통합)
- 필터: 태그 멀티셀렉트, 정렬(인기/최신/랜덤)
- 무한 스크롤 또는 페이지네이션

### 상세 (/p/[id])
- 제목 + 설명 (마크다운 렌더링)
- Prompt Builder 폼 (변수 자동 렌더링)
- 완성 프롬프트 미리보기 (실시간 치환)
- 액션: Generate, Copy, Chat UI로 사용, 북마크

### Prompt Builder 동작
- 실시간 치환: 입력할 때마다 미리보기 갱신
- Generate 버튼: 서버 API 호출하여 usage_events 기록
- 필수 변수 표시: * 또는 (필수) 라벨
