# 에러 코드 표준 (ERROR_CODES)

## 응답 형식

```json
{
  "code": "AUTH_001",
  "message": "인증 토큰이 유효하지 않습니다.",
  "details": {}
}
```

## 에러 코드 목록

### 인증/인가 (AUTH)

| HTTP | Code | Message |
|------|------|---------|
| 401 | AUTH_001 | 인증 토큰이 없거나 유효하지 않습니다 |
| 401 | AUTH_002 | 토큰이 만료되었습니다 |
| 403 | AUTH_003 | 접근 권한이 없습니다 |
| 403 | AUTH_004 | 관리자 권한이 필요합니다 |

### 프롬프트 (PROMPT)

| HTTP | Code | Message |
|------|------|---------|
| 404 | PROMPT_001 | 프롬프트를 찾을 수 없습니다 |
| 400 | PROMPT_002 | 필수 변수가 입력되지 않았습니다 |
| 400 | PROMPT_003 | 템플릿 형식이 올바르지 않습니다 |
| 409 | PROMPT_004 | 이미 북마크된 프롬프트입니다 |

### 요청 검증 (VALID)

| HTTP | Code | Message |
|------|------|---------|
| 400 | VALID_001 | 필수 파라미터가 없습니다 |
| 400 | VALID_002 | 파라미터 형식이 올바르지 않습니다 |
| 422 | VALID_003 | 데이터 검증 실패 |

### 레이트리밋 (RATE)

| HTTP | Code | Message |
|------|------|---------|
| 429 | RATE_001 | 요청 한도를 초과했습니다 (일일 제한) |
| 429 | RATE_002 | 요청 한도를 초과했습니다 (시간당 제한) |

### 서버 오류 (SRV)

| HTTP | Code | Message |
|------|------|---------|
| 500 | SRV_001 | 서버 내부 오류가 발생했습니다 |
| 503 | SRV_002 | 서비스를 일시적으로 사용할 수 없습니다 |
